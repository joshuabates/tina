# Border Radius

All radii derive from a single `--radius` CSS custom property (0.5rem / 8px).

## Variants

## How It Works

The base token `--radius: 0.5rem` feeds the scale:

| Token | Tailwind | Calculation | Result |
|-------|----------|-------------|--------|
| `--radius` | `rounded-lg` | `var(--radius)` | 8px |
| — | `rounded-md` | `calc(var(--radius) - 2px)` | 6px |
| — | `rounded-sm` | `calc(var(--radius) - 4px)` | 4px |

Changing `--radius` shifts the entire scale proportionally.
