# Spacing

Uses Tailwind's default spacing scale. Values work with `p-`, `m-`, `gap-`, `w-`, `h-`, and other spacing utilities.

## Usage

| Utility | Example | Description |
|---------|---------|-------------|
| `p-{n}` | `p-4` | Padding (16px) |
| `m-{n}` | `m-2` | Margin (8px) |
| `gap-{n}` | `gap-3` | Flex/grid gap (12px) |
| `w-{n}` | `w-6` | Width (24px) |
| `h-{n}` | `h-8` | Height (32px) |
| `space-x-{n}` | `space-x-2` | Horizontal spacing (8px) |
